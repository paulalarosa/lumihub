# Como Criar um Cliente

## Passo a Passo

1. **Acesse a Área de Clientes**
   - No menu lateral esquerdo, clique em "Clientes".
   - Alternativamente, pressione `Cmd+K` (ou `Ctrl+K` no Windows) e digite "Novo Cliente".

2. **Inicie o Cadastro**
   - Clique no botão "Novo Cliente" localizado no canto superior direito da tela.

3. **Preencha os Dados do Cliente**
   Os seguintes campos são obrigatórios:
   - **Nome Completo**: O nome que será usado em contratos e comunicações.
   - **Email**: Principal canal de contato e login (se aplicável).
   - **Telefone**: Importante para contato via WhatsApp. Use o formato com DDD (ex: 11999999999).

   Campos opcionais recomendados:
   - **CPF**: Essencial para a geração automática de contratos válidos.
   - **Endereço**: Utilizado na cláusula de qualificação dos contratos.
   - **Data de Nascimento**: Para ações de marketing e relacionamento.

4. **Salvar**
   - Clique em "Salvar Cliente".
   - O sistema criará o registro e você será redirecionado para a ficha do cliente.

## Dicas
- O sistema valida automaticamente se o email já está cadastrado.
- Se você tiver um contrato em PDF assinado, pode fazer o upload diretamente no momento do cadastro.
