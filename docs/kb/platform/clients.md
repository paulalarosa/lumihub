# Gestão de Clientes

O CRM do Khaos Kontrol centraliza todas as informações dos seus clientes.

## Ficha do Cliente
Cada cliente possui uma ficha técnica contendo:
- **Dados Pessoais**: Nome, Contatos, Endereço.
- **Projetos Vinculados**: Histórico de contratos e eventos.
- **Financeiro**: Total gasto, pendências e histórico de pagamentos.
- **Documentos**: Upload de arquivos (RG, Comprovante de Residência).

## Importação/Exportação
Atualmente a importação é feita via planilha CSV (contate o suporte para ajuda). A exportação de dados está disponível no botão "Exportar" na lista de clientes.
