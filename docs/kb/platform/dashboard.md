# Dashboard (Painel de Controle)

A Dashboard é a tela inicial do Khaos Kontrol, projetada para dar uma visão imediata da saúde do seu negócio.

## Seções Principais

### 1. Métricas Financeiras
- **Receita Mensal**: Total faturado no mês atual.
- **Projeção**: Previsão de faturamento baseada nos contratos fechados para os próximos meses.
- **Pendentes**: Valores a receber de clientes em atraso ou próximos do vencimento.

### 2. Agenda Rápida
- Mostra os próximos 3 eventos ou compromissos.
- Clique em "Ver tudo" para ir à agenda completa.

### 3. Assistente Lumi IA
- O terminal de acesso à Lumi IA fica disponível aqui.
- Você pode perguntar "Quais meus compromissos de hoje?" ou "Crie uma tarefa" diretamente por ali.

### 4. Atalhos
- Acesso rápido para: Novo Cliente, Novo Projeto, Ver Relatórios.

## Personalização
- Atualmente, o layout da dashboard é fixo para garantir a melhor experiência de usabilidade baseada em prioridades de gestão de eventos.
